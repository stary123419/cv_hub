# GitHub Labels Setup

GitHub не позволяет создавать labels через файл в репозитории — их нужно добавить вручную.

Вот список labels для CV Hub:

---

## Как добавить

Перейди на страницу:
```
https://github.com/ТОЙ_АККАУНТ/cv-hub/labels
```

Нажми **New label** и создай следующие:

---

## Labels

| Name | Color | Description |
|------|-------|-------------|
| `bug` | `#d73a4a` | Something is broken |
| `enhancement` | `#a2eeef` | New feature or improvement |
| `documentation` | `#0075ca` | Improvements to docs |
| `roadmap` | `#6f42c1` | Planned feature from roadmap |
| `help wanted` | `#008672` | Open for contributions |
| `good first issue` | `#7057ff` | Good for newcomers |
| `question` | `#d876e3` | Further information requested |
| `wontfix` | `#ffffff` | Out of scope for this project |

---

После того как labels созданы — они будут подставляться автоматически из issue templates.
